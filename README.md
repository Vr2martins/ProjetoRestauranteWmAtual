# Restaurante WM - Projeto Web

## Estrutura do Projeto

O projeto do Restaurante WM está organizado da seguinte forma:

```
/
├── assets/                  # Recursos estáticos
│   ├── images/              # Imagens do site
│   ├── css/                 # Arquivos de estilo
│   └── js/                  # Scripts JavaScript
│
├── pages/                   # Páginas do site
│   ├── home/                # Página inicial
│   ├── menu/                # Cardápio
│   ├── about/               # Página Sobre
│   └── auth/                # Páginas de autenticação (login/registro)
│
└── README.md                # Documentação do projeto
```

## Navegação

- **Página Inicial**: Apresentação do restaurante e informações principais
- **Cardápio**: Lista de pratos e preços oferecidos pelo restaurante
- **Sobre**: Informações sobre a história e localização do restaurante
- **Login/Registro**: Área de autenticação para usuários

## Desenvolvimento

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript, seguindo boas práticas de organização de código e estrutura de arquivos.
